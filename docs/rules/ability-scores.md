# Ability Scores

## Rolling Stats

Roll **3d6** for each of the six stats **in order** (no rearranging).
If no score exceeds 14, you may reroll all six.

## The Six Stats

| Stat | Abbreviation | Primary Uses |
|---|---|---|
| Strength     | STR | Melee attack rolls; gear slot capacity |
| Dexterity    | DEX | Ranged attack rolls; unarmored AC (10 + DEX mod); initiative |
| Constitution | CON | HP at 1st level (added once, never again); death timer modifier |
| Intelligence | INT | Wizard spellcasting rolls; stabilization checks; bonus languages |
| Wisdom       | WIS | Priest spellcasting rolls |
| Charisma     | CHA | Social interactions; reaction rolls |

## Modifier Table

| Score | Modifier |
|---|---|
| 1–3   | −4 |
| 4–5   | −3 |
| 6–7   | −2 |
| 8–9   | −1 |
| 10–11 |  0 |
| 12–13 | +1 |
| 14–15 | +2 |
| 16–17 | +3 |
| 18+   | +4 |

## Key Rules to Remember

- STR and DEX modifiers add to **attack rolls only** — never to damage rolls.
- CON modifier applies to HP at **1st level only**; it is not re-added on each level-up.
- Damage bonuses come exclusively from Talents or magic items.
- Stabilizing a dying ally requires a **DC 15 INT check** (see death-and-dying.md).
