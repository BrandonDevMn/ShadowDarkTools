# Core Resolution Mechanic

## The d20 Check

Roll **1d20 + relevant ability modifier** and compare the total to a **Difficulty Class (DC)**.
Meeting or beating the DC is a success. All checks — ability checks, attack rolls, spellcasting rolls — use this same mechanic.

## Standard DCs

| DC | Difficulty |
|---|---|
| 9  | Easy        |
| 12 | Moderate    |
| 15 | Hard        |
| 18 | Very Hard   |

## Attack Rolls

- **Melee:** 1d20 + STR modifier + any Talent bonuses vs. target's AC
- **Ranged:** 1d20 + DEX modifier + any Talent bonuses vs. target's AC
- STR and DEX modifiers add to attack rolls **only** — not to damage

## Advantage and Disadvantage

When a rule grants **advantage**, roll 2d20 and take the **higher** result.
When a rule imposes **disadvantage**, roll 2d20 and take the **lower** result.
Applies to any d20 roll: checks, attacks, or spellcasting.

## No Skill List

There are no trained skills. All checks use the relevant ability modifier directly.
Class and ancestry features grant advantage on specific check types instead (e.g., Thief Thievery, Fighter Grit).

## What Does Not Exist

Shadowdark deliberately omits these 5e constructs:
- Opportunity attacks
- Bonus actions
- Reactions
- Proficiency bonus
