# ShadowDarkTools — Claude Instructions

Before doing any work in this repository, read every file in the `/docs/` folder.
Those files define platform constraints, coding conventions, and the collaboration
process that must be followed for all changes.

**Never commit secrets** — API keys, tokens, passwords, private keys, or `.env` files must never be staged or committed. See `docs/how-we-work.md` for the full policy.

Current docs to read at startup:
- `docs/how-we-work.md` — workflow, naming, comments, testing, and git rules
- `docs/mobile-web-app.md` — iPhone XS Max / Safari / GitHub Pages constraints
